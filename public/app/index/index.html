If you have Node.js installed and would like to follow along, run the following commands in your local repository directory:

  1. git remote add upstream https://github.com/icfi-tech-crunch-munch/wcag-in-my-508.git
  2. git pull upstream master
  3. npm install
  4. bower install
  5. npm start

Then visit http://localhost:3000
